# rum-diary-db-mongo

Mongo database adapter for rum-diary.org

## Prerequisites:

* [nodejs](http://nodejs.org/) &gt;= 0.10.0
* [mongodb](http://www.mongodb.org/) - on OSX with homebrew - `brew install mongo`

## Tests
`npm test`
